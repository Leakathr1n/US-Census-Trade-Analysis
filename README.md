# US Census Trade Analysis

This repository contains a collection of code that we used to analyse some US Census Trade data in preparation for the UN DESA Monthly Briefing: 

Rokosz, K. (2025). The outlook for international trade amid structural shifts and rising restrictions (No. 188; UN DESA Monthly Briefings). UN Department of Economic and Social Affairs. https://www.un.org/development/desa/dpad/publication/world-economic-situation-and-prospects-march-2025-briefing-no-188/

This analysis contains to main streams: 

## 1. Analysing the Top 5 HS codes per importer: 
The importing country can be specified by the researcher, and the code will automatically produce the Top 5 HS codes. The relevant scripts are *02 Scripts/1. Importing Monthly US Commodities Imports.R* to 02 *Scripts/4. Creating Stacked Bar Charts.R* with outputs of the resulting code being in *03 Outputs/4. Stacked Bar Charts per HS Code*.

## 2. A brief overview of US trade deficit and how it has developed from 2013 to 2024: 
The relvant scripts are *02 Scripts/5. Importing US Trade Deficit 2013-2024.R* to *02 Scripts/6. Preparing US Trade Deficit 2013-2024.R* with output being *03 Outputs/6.6. US Trade Deficit 2013-2024.xlsx*.
